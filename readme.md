# Devcamper API

> Backend API for the devcamper applicaton, which is a bootcamp directory website.

## Usage

Rename "config/sample.env" to "config/config.env" and update values/settings to your own

## Install dependencies

```
npm install
```

## Run app

```
# Run in dev mode
npm run dev

# Run in prod mode
npm start
```
